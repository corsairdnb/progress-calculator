Градусо-сутки отопительного периода для г. Санкт-Петербург определены по формуле:
ГСОП (градусо-сутки отопительного период) = (tв - tот.пер) · Zот.пер = (20 - (-1.8)) · 220 = 4536,9;
где: tв  =  t° периода со средней суточной температурой воздуха (≤ 8 °С) = -1.8 °С ;
Zот.пер  =  длительность периода со средней суточной температурой воздуха (≤ 8 °С) = 220 (сут.)
Методом линейной интерполяции определяем базовый уровень удельного годового расхода электрической энергии на общедомовые нужды для 5-ти этажного дома
256+(6-5)*(263-256)/(6-4) = 259,5
Методом линейной интерполяции определяем базовый уровень удельного годового расхода электрической энергии на общедомовые нужды для 5-ти этажного дома при ГСОП равном 4536,9 °С*сут/год
234+(5000-4536,9)*(256-234)/(5000-4000) = 244,188
Методом линейной интерполяции вычисляем среднее значение (259,5+244,188)/2 = 251,844
Дом по адресу не оборудован лифтом следовательно из базового уровня вычитаем 3 кВт*ч/м2. Базовый показатель за вычетом лифта = 251,44 - 3 = 248,844
